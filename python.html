séance 1

l Python est langage de programmation haut niveau, il est poppulaire et utilisé par la CIA, la NASA, Disney et Google...
consultez la page "présentation avec choaib" mon ami vous donera plus informations, bon ! faisant nos premiers pas avec ce langage, mais avant tout, assurez vous de télécharger et de installer Python 3.x et non le 2, et lors de l'instalation, n'oubliez pas de cocher "intall pip", car c'est avec ce petit bijou qu'on pourra installer les extensions de Python, dont Kivy qu'on vera pour la partie developement mobile.
maintenant que Python est installé configurez votre environnement de travail : un éditeur de texte lié avec l'interpreteur Python (oui c'est ce que vous venez d'installer), de sorte que le programme s'execute à partir du code sur l'éditeur, bon faite de recherche pour savoir comment :p.
testons si le tout est bien configuré, si vous utilisez l'éditeur de Python utilisez f5 pour lancer vos programmes, si vous utilisez sublime text utiliser f7.
code :
print("test")

si le mot test s'affiche sur la console c'est que tout va bien on se voit à la proochaine séance :).
