séance 1

vous pouvez automatiser vos tâches sur windows plus facilement avec batch, ceci est possible seulement avec un éditeur de text comme NotePad par exemple.
le fichier créé "test.txt" il faut changer .txt en .bat, choisir modifier apres un clic droit sur ce dernier et voilà ! une page blanche est à notre disposition.
commençant par quelque chose de simple, un programme qui vous appelle par votre nom.
code :
echo HE !!! monsef !

enregistrez ce que venez d'écrire avec "ctrt+s", ouvrez votre fichier .bat ici dans cet exemple "test.bat".
le programme s'ouvre et se ferme rapidement, d'accord on va le "coincer" avec la fonction "pause"
code :
echo HE !!! monsef !
pause

encore une fois enregistrez et ouvrez, ca marche mais il y a du texte indésirable dans le prog.
d'accord d'accord ! on va l'enlevé avec "@echo off" au début du programme.
code :
@echo off
echo HE !!! monsef !
pause

exercice :
changer ce programme pour qui vous appelle par votre nom.
