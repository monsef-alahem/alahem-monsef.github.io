commandes bash:


cd : pour changer le dossier de travail : cd /usr/user_name/Images
mkdir : pour créer un dossier ex : mkdir dossier1
locate : pour chercher un fichier ou dossiers sur sa machine ex : locate Bureau
cp : pour copier ex : cp photo1 /usr/monsef/Bureau
mv : pour deplacer ex : mv photo1 /usr/monsef/Bureau
rm : pour suprimer ex : rm photo1
at : pour lancer une commande à un moment donné ex :
     at 15:47
     at 15:47 tomorrow
     at 15:47 17/11/2020
     at 15:47 now +36 minutes
     at 15:47 now +2 months
     at 15:47 now +1 years
     at 15:47 now +6 hours
     tapez la liste de commandes que vous voulez puis apuiyez sur "ctrl+d" pour fermer at.
fdisk -l : pour afficher vos périphériques de stockage et leur informations ex : fdisk -l
apt-get : pour installer et modifier des programmes, et c'est plus prtique que windows ex : sudo apt-get install sublime
