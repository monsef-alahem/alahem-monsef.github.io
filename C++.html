C++

bon alors là, le C++ c'est une version améliorée de C (d'où le nom, car "++" signifie incrémentation) mais à part le concept POO programmation orientée objet et des nouvelles fonctions, C et C++ sont assez similaires

compiler un fichier C++ avec CMD (notez que appart g++ et .cpp c'est la même chose que C):

g++ -o my_exe.exe my_c++_file.cpp

pour les phases de compilations jettez un coup d'oeuil à la page C.

exemples de compilation avec C++

note : g++ doit etre accecible a travers CMD, pour en être sûre, ouvrez la console aver le raccourci window+R, tapez cmd et validez.

On teste ? tapez g++, si la commande est introuvable SDL :

g++ -c --std=c++11 *.cpp
g++ -o myapp.exe *.o -lSDLmain -lSDL -lSDL_image -lSDL_mixer -lSDL_ttf
start myapp.exe"

C++ 2011 & SDL2 :

g++ -c --std=c++11 *.cpp
g++ -o myapp.exe *.o -lSDL2main -lSDL2 -lSDL2_image -lSDL2_mixer -lSDL2_ttf
start myapp.exe"

SFML :

g++ -c --std=c++11 *.cpp
g++ -o myapp.exe *.o -lsfml-graphics -lsfml-window -lsfml-system
start myapp.exe"

openGL avec SDL2 :

g++ -c *.cpp
g++ -o myapp.exe *.o -lSDL2main -lSDL2 -lglu32 -lopengl32 -lglew32
start myapp.exe"

Windows (pour manipuler le système :O) :

g++ -c *.cpp
g++ -o myapp.exe *.o -lgdi32 -luser32 -lkernel32 -lPsapi
start myapp.exe"
