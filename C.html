C

compiler un fichier C avec CMD:

gcc -o my_exe.exe my_c_file.c

les quatre phases de compilation :

1- le préprocessing
2- la compilation
3- l'assemblage
4- la liaison

exemples de compilation avec C

note : gcc ou votre compilateur C doit être accessible à travers CMD

SDL :

gcc -c *.c
gcc -o myapp.exe *.o -lSDLmain -lSDL -lSDL_image -lSDL_mixer -lSDL_ttf
start myapp.exe"

SDL2 :

gcc -c *.c
gcc -o myapp.exe *.o -lSDL2main -lSDL2 -lSDL2_image -lSDL2_mixer -lSDL2_ttf
start myapp.exe"

C & pthread & socket :

gcc -c *.c
gcc -o myapp.exe *.o -lpthread -lws2_32
start myapp.exe"
